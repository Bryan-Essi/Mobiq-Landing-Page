# MOBIQ Landing Page

Landing page React + Vite for MOBIQ, including all media assets (images and demo videos).

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```

## Project structure

- `src/pages/Presentation.tsx`: landing page component
- `src/pages/Presentation.css`: landing page styles
- `src/assets/presentation/`: image assets
- `public/1.mp4`, `public/watch-demo.mp4`: demo videos
