ADB Framework Telco Automation
A comprehensive telecommunications testing framework for Android devices using ADB.

This software includes:
- FastAPI backend server
- React frontend interface  
- Electron desktop application
- 29 telecommunications test modules
- Multi-device orchestration capabilities
- Real-time monitoring and reporting

For support and documentation, visit:
https://F2G-Telco-Academy/ADB-automation-tool
